How to point DNS to your server
  1. Log in to your DNS provider
  2. Add an A record: * → YOUR_SERVER_IP
  3. Add an A record: @ → YOUR_SERVER_IP
  4. Wait 1–5 minutes for DNS propagation

Cloudflare API Token

Used by Caddy to automatically issue SSL certificates via DNS.

How to create a Cloudflare API token
  1. Go to dash.cloudflare.com → My Profile → API Tokens
  2. Click Create Token
  3. Use the Edit zone DNS template
  4. Set Zone Resources to your domain
  5. Click Continue → Create Token

Tailscale Auth Key

Required for secure mesh networking.

How to generate a Tailscale auth key
  1. Go to tailscale.com/admin/settings/keys
  2. Click Generate auth key
  3. Enable Reusable and Ephemeral (optional)
  4. Copy the key
