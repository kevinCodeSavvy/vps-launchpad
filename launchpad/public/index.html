Create Admin User

Running as root permanently is risky. This creates a dedicated non-root user for day-to-day server management, then locks down SSH so only key-based login is allowed.

Domain Name

Each service gets its own subdomain (e.g. search.yourdomain.com, paperclip.yourdomain.com). Caddy will automatically issue SSL certificates for all of them — no manual cert management needed.

How to point DNS to your server
  1. Log in to your DNS provider
  2. Add an A record: * → YOUR_SERVER_IP
  3. Add an A record: @ → YOUR_SERVER_IP
  4. Wait 1–5 minutes for DNS propagation

Cloudflare API Token

Caddy uses this token to prove domain ownership to Let's Encrypt via DNS challenge — meaning SSL certificates are issued automatically, even for subdomains, without any ports needing to be open during setup.

How to create a Cloudflare API token
  1. Go to dash.cloudflare.com → My Profile → API Tokens
  2. Click Create Token
  3. Use the Edit zone DNS template
  4. Set Zone Resources to your domain
  5. Click Continue → Create Token

Tailscale

Required for secure mesh networking.

Tailscale creates a private, encrypted network between your devices. Once connected, you can reach your server and all its services securely from anywhere — your phone, laptop, or home network — without exposing anything to the public internet.

How to generate a Tailscale auth key
  1. Go to tailscale.com/admin/settings/keys
  2. Click Generate auth key
  3. Enable Reusable and Ephemeral (optional)
  4. Copy the key
